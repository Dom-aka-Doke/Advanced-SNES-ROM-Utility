SwapBin ROM

Swaps binaries to get a "Close-to-SNES-Mask-ROM"-layout.

Use 27C801 EPROM(s) and rewire pins as followed:

27C801 -> Nintendo Mask ROM
24     ->     31
31     ->     24

Deinterleave

Some copy stations like "Game Doctor SF" or "Super UFO" use interleaving when dumping HiROM games. This function will reverse this process. Please use ROMs with copy station header, unless you know which copy station was used!

go back

Interleave

Some patches or copy stations require interleaved ROMs. This function will interleave your ROM. If you have interleaved a ROM, patched it and want to play it in an emulator or flashcard, don't forget to deinterleave and fix checksum again!

Note: Only HiROMs can be interleaved!

Remove SRAM Checks

Some games have a copy protection preventing the ROM from running, when cartridges have more SRAM than needed (copy stations/some cheap flashcards/deprecated emulators). Use this function to remove most known SRAM checks.

Options

Miror ROM on expanding

  • Checked: ROMs' content gets mirrored when it gets expanded.
  • Unchecked: Standard expanding will be perfomed to get more space available.

Scan protections on loading

  • Checked: ROM will be scanned instantly for copy protections on loading. This might take a few seconds.
  • Unchecked: You have to manually click the Scan button to check for copy protections.

Commandline (CLI)

Use Windows Commandline (CMD) for executing batch operations (or just for processing a single file, if you prefer using the commandline).

Build your own command chain to execute operations in the order you specified!

Notes:

  • Deinterleaving and interleaving might require user action! This will bring up a windows form and can only work within a Windows environment

  • All paths must be in absolute format

    • Example:

    Advanced_SNES_ROM_Utility -path "C:\ROMs" -recursive -header add -patch "C:\Patches\patch.ips" -header remove -fixromsize -fixchecksum

    This command will add a header to all ROMs in C:\ROMs (subfolders included), apply an IPS patch, remove header afterwards, fix internal ROM size and checksum and save the modfied file with an extension.

    Usage:    
    Advanced_SNES_ROM_Utility [options]
     
    Options:
      -path <FILEPATH, FOLDER> Path or folder of the ROM file(s)
      -recursive Include files from subdirectories
      -overwrite Overwrite files on saving (else an extension will be used)
      -header <add, remove> Adds or removes a header
      -fixchecksum Fixes a broken checksum
      -fixromsize Fixes a wrong internal ROM size
      -removeregion Removes existing region checks
      -removesram Removes existing SRAM checks
      -removeslowrom Removes existing SlowROM checks
      -deinterleave Deinterleave, if ROM is in interleaved format
      -interleave Interleave, if ROM is in deinterleaved format
      -patch <FILEPATH> Patch ROM with a specific patch
      -log Write output to a logfile
      -version Show version information
      -help Show help and usage information
